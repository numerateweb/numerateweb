numerateweb

Mathematics on the Web of Data.

The numerateweb project is an effort to bridge the gap between linked data and computer algebra or simulation systems.

Math vocabulary

The numerateweb math ontology provides an OpenMath inspired vocabulary to represent mathematical expressions as linked data.

View details »

Rule language

The numerateweb rule language connects mathematical expressions to classes and properties of ontologies. This allows automatic computation of numerical property values or constraint checking.

Tools

To bootstrap the creation and consumption of numerateweb resources a initial set of tools will be provided.
